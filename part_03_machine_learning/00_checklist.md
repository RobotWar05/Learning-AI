# Checklist Machine Learning

- [ ] Phân biệt supervised và unsupervised learning.
- [ ] Phân biệt regression và classification.
- [ ] Hiểu feature, label, sample, dataset.
- [ ] Hiểu train/test split.
- [ ] Hiểu overfitting và underfitting.
- [ ] Biết accuracy dùng cho classification.
- [ ] Biết MSE/MAE dùng cho regression.
- [ ] Nắm quy trình `fit`, `predict`, `evaluate`.
- [ ] Biết khi nào cần chuẩn hóa dữ liệu.
- [ ] Biết ý nghĩa confusion matrix.
