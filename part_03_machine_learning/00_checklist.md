# Checklist Machine Learning

## Lý thuyết ML

- [ ] Phân biệt AI, ML, DL.
- [ ] Hiểu supervised learning.
- [ ] Hiểu unsupervised learning.
- [ ] Biết reinforcement learning ở mức khái niệm.
- [ ] Phân biệt regression, classification, clustering.
- [ ] Hiểu loss function.
- [ ] Hiểu gradient descent ở mức trực giác.
- [ ] Hiểu bias/variance.

## Chuẩn bị dữ liệu

- [ ] Xác định feature `X`.
- [ ] Xác định label `y`.
- [ ] Chia train/test.
- [ ] Biết khi nào cần validation set.
- [ ] Biết khi nào cần feature scaling.
- [ ] Kiểm tra missing values và dữ liệu lệch nhãn.

## Supervised learning

- [ ] Linear Regression.
- [ ] Logistic Regression.
- [ ] KNN.
- [ ] Decision Tree.
- [ ] Random Forest.
- [ ] Naive Bayes.
- [ ] SVM ở mức ý tưởng.

## Unsupervised learning

- [ ] K-means.
- [ ] Hiểu clustering không cần label.
- [ ] Biết inertia ở mức khái niệm.

## Metrics

- [ ] Accuracy.
- [ ] Precision.
- [ ] Recall.
- [ ] F1-score.
- [ ] Confusion matrix.
- [ ] MSE.
- [ ] MAE.
- [ ] RMSE.

## Scikit-learn workflow

- [ ] Import model.
- [ ] Tạo model.
- [ ] `fit(X_train, y_train)`.
- [ ] `predict(X_test)`.
- [ ] Đánh giá bằng metric.
- [ ] Dùng `cross_val_score`.
