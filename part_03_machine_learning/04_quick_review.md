# Ôn nhanh Machine Learning

| Khái niệm | Ý nghĩa ngắn |
|---|---|
| Feature | Biến đầu vào |
| Label | Giá trị cần dự đoán |
| Train set | Dữ liệu dùng để học |
| Test set | Dữ liệu dùng để đánh giá |
| Classification | Dự đoán nhãn |
| Regression | Dự đoán số liên tục |
| Overfitting | Học quá sát train data |
| Metric | Thước đo chất lượng model |
