# Phần 3: Machine Learning

## Mục tiêu

Hiểu quy trình Machine Learning, các thuật toán cơ bản và cách dùng scikit-learn ở mức phục vụ bài thi.

## Kiến thức sẽ học

- Supervised learning và unsupervised learning.

- Train/test split.
- Overfitting và underfitting.
- Loss, metric, accuracy, precision, recall, F1.
- Linear Regression, Logistic Regression, Decision Tree, KNN, K-Means.
- Quy trình code scikit-learn.
