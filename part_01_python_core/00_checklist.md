# Checklist Python Core

## Cú pháp cơ bản

- [ ] Biết dùng `print()`.
- [ ] Biết tạo biến và gán giá trị.
- [ ] Phân biệt `int`, `float`, `str`, `bool`.
- [ ] Biết dùng `type()`.
- [ ] Biết ép kiểu bằng `int()`, `float()`, `str()`.
- [ ] Biết toán tử `+`, `-`, `*`, `/`, `//`, `%`, `**`.
- [ ] Biết toán tử so sánh `==`, `!=`, `>`, `<`, `>=`, `<=`.

## Điều kiện và vòng lặp

- [ ] Viết được `if/elif/else`.
- [ ] Hiểu dấu `:` sau điều kiện.
- [ ] Hiểu indentation quyết định block code.
- [ ] Dùng được `for item in list`.
- [ ] Dùng được `for i in range(...)`.
- [ ] Dùng được `while` với điều kiện dừng rõ ràng.
- [ ] Tránh vòng lặp vô hạn.

## List/Tuple/Dict/Set

- [ ] Tạo và truy cập phần tử list bằng index.
- [ ] Dùng `append`, `remove`, `len`, `max`, `min`, `sum`.
- [ ] Tính trung bình list số.
- [ ] Duyệt list bằng vòng lặp.
- [ ] Viết list comprehension đơn giản.
- [ ] Hiểu tuple là dạng gần như không đổi.
- [ ] Tạo dict và lấy value theo key.
- [ ] Kiểm tra key có tồn tại trong dict.
- [ ] Dùng set để loại phần tử trùng.

## String

- [ ] Dùng `strip()` để bỏ khoảng trắng hai đầu.
- [ ] Dùng `lower()` để chuẩn hóa chữ thường.
- [ ] Dùng `split()` để tách chuỗi.
- [ ] Dùng `join()` để ghép chuỗi.
- [ ] Dùng f-string để format output.

## Function

- [ ] Viết function có tham số.
- [ ] Biết khác nhau giữa `print` và `return`.
- [ ] Viết function kiểm tra chẵn/lẻ.
- [ ] Viết function tính trung bình.
- [ ] Viết function xử lý list hoặc string.

## File và Exception

- [ ] Biết ý nghĩa `with open(...) as f`.
- [ ] Biết mode `r`, `w`, `a`.
- [ ] Biết dùng `try/except` để tránh crash chương trình.
- [ ] Biết lỗi `ValueError`, `KeyError`, `IndexError` ở mức cơ bản.

## Sẵn sàng cho AI

- [ ] Đếm số lượng nhãn trong dataset nhỏ.
- [ ] Tính tỷ lệ nhãn.
- [ ] Chuẩn hóa min-max cho list số.
- [ ] Chia dữ liệu train/test đơn giản.
- [ ] Hiểu dữ liệu dạng `list[dict]`.
