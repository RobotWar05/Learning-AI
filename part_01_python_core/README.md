# Phần 1: Python nền tảng

## Mục tiêu

Sau phần này, bạn cần đọc hiểu và viết được các đoạn code Python cơ bản phục vụ AI/ML:

- Biến, kiểu dữ liệu, toán tử.
- `if/elif/else`, `for`, `while`, `range`.
- `list`, `tuple`, `dict`, `set`.
- Xử lý chuỗi.
- Viết function.
- Đọc/ghi file ở mức cơ bản.
- Bắt lỗi bằng `try/except`.
- Làm các tác vụ nhỏ gần AI: đếm nhãn, chuẩn hóa min-max, chia train/test.

## Kiến thức cần học

1. Cú pháp Python cơ bản.
2. Điều kiện và vòng lặp.
3. Cấu trúc dữ liệu.
4. Function và `return`.
5. File và exception.
6. Bài toán dữ liệu nhỏ phục vụ AI.

## Thứ tự học

1. Đọc `00_checklist.md`.
2. Gõ lại `01_must_memorize_code.py`.
3. Chạy file code và quan sát output.
4. Làm bài trong `02_exercises.md`.
5. Ghi lỗi sai vào `03_notes.md`.
6. Ôn nhanh bằng `04_quick_review.md`.

## Dạng bài thi hay gặp

- Dự đoán output của đoạn code.
- Viết vòng lặp tính tổng, trung bình, min, max.
- Đếm tần suất phần tử trong list.
- Xử lý string bằng `split`, `join`, `lower`, `strip`.
- Viết function có `return`.
- Sửa lỗi cú pháp hoặc logic đơn giản.
- Đọc list dữ liệu nhỏ và đếm nhãn.
