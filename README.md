# Learning-AI

Repository này là lộ trình học AI bằng tiếng Việt, phục vụ ôn thi cấp tốc nhưng vẫn giữ tư duy học có hệ thống. Nội dung tập trung vào phần thường gặp trong bài thi: lý thuyết AI/ML/DL, đọc hiểu code Python, xử lý dữ liệu, pipeline RAG, thiết kế chatbot và tư duy monitoring/ops cho hệ thống AI.

Trạng thái repo: **Đang phát triển đầy đủ toàn bộ 6 phần**.

## Bản đồ 6 phần học

| Phần | Tên phần | Mục tiêu | Trạng thái |
|---|---|---|---|
| Phần 1 | Python nền tảng | Đọc hiểu và viết code Python cơ bản phục vụ AI | Hoàn thiện |
| Phần 2 | NumPy và Pandas | Xử lý dữ liệu dạng mảng và bảng | Hoàn thiện bản học cấp tốc |
| Phần 3 | Machine Learning | Hiểu thuật toán ML và code scikit-learn | Hoàn thiện bản học cấp tốc |
| Phần 4 | Deep Learning | Hiểu neural network, loss, optimizer, PyTorch cơ bản | Hoàn thiện bản học cấp tốc |
| Phần 5 | RAG và Chatbot | Hiểu pipeline RAG và thiết kế chatbot | Hoàn thiện bản học cấp tốc |
| Phần 6 | Agent Monitoring/Ops | Hiểu logs, metrics, traces, evaluation, deployment mindset | Hoàn thiện bản học cấp tốc |

## Cách học từng phần

1. Đọc `README.md` của phần đó để hiểu mục tiêu và thứ tự học.
2. Tick kiến thức trong `00_checklist.md`.
3. Đọc notes lý thuyết trước khi chạy code.
4. Với file `.py`, đọc file `.md` giải thích đi kèm rồi tự gõ lại code.
5. Làm bài tập, sau đó dùng quick review để tự kiểm tra.
6. Cập nhật tiến độ trong [progress_tracker.md](progress_tracker.md).

## Quy tắc học

- Không học thuộc định nghĩa rời rạc; luôn hỏi: input là gì, output là gì, metric là gì.
- Với code Python, phải tự gõ lại ít nhất một lần.
- Với ML/DL, phải phân biệt được bài toán classification, regression, clustering.
- Với RAG/chatbot, phải nắm pipeline dữ liệu trước khi học tool.
- Với monitoring/ops, phải biết cần log gì, đo gì, cảnh báo gì.
- Nếu gặp thuật ngữ mới, ghi lại trong notes của phần tương ứng.

## File điều hướng

- [progress_tracker.md](progress_tracker.md): bảng theo dõi tiến độ từng file.
