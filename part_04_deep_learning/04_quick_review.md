# Ôn nhanh Deep Learning

| Khái niệm | Ý nghĩa ngắn |
|---|---|
| Tensor | Mảng số nhiều chiều |
| Layer | Tầng xử lý dữ liệu |
| Weight | Tham số học được |
| Bias | Hệ số lệch |
| Activation | Hàm phi tuyến |
| Loss | Sai số cần giảm |
| Optimizer | Cách cập nhật tham số |
| Epoch | Một lượt học qua train set |

## Câu hỏi nhanh

1. Vì sao neural network cần nhiều layer?
2. ReLU thường dùng để làm gì?
3. `loss.backward()` có ý nghĩa gì trong PyTorch?
4. Vì sao phải gọi `optimizer.zero_grad()`?
5. Overfitting trong deep learning biểu hiện ra sao?
