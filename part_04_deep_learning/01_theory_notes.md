# Ghi chú lý thuyết Deep Learning

## Bản chất

Deep Learning dùng nhiều layer để học biểu diễn dữ liệu. Model không chỉ học luật trực tiếp, mà học các tầng đặc trưng trung gian.

## Thành phần chính

- Input: dữ liệu đưa vào model.
- Layer: phép biến đổi dữ liệu.
- Activation: tạo khả năng học quan hệ phi tuyến.
- Loss: đo model sai bao nhiêu.
- Optimizer: cập nhật weight để giảm loss.

## Rủi ro cần nhớ

- Learning rate quá cao có thể làm training không ổn định.
- Model quá lớn dễ overfit.
- Dữ liệu ít hoặc nhiễu làm kết quả kém.
