# Checklist Deep Learning

## Neural network theory

- [ ] Hiểu neuron nhận input và tạo output.
- [ ] Hiểu weight và bias.
- [ ] Hiểu dense layer.
- [ ] Biết activation function tạo phi tuyến.
- [ ] Phân biệt ReLU, sigmoid, softmax.
- [ ] Hiểu forward propagation.
- [ ] Hiểu backpropagation ở mức trực giác.

## Tensors và layers

- [ ] Tạo tensor.
- [ ] Xem tensor shape.
- [ ] Hiểu batch dimension.
- [ ] Dùng `nn.Linear`.
- [ ] Dùng `nn.Sequential`.
- [ ] Viết `nn.Module` cơ bản.

## Loss và optimizer

- [ ] Biết MSE dùng cho regression.
- [ ] Biết CrossEntropy dùng cho multi-class classification.
- [ ] Biết optimizer cập nhật tham số.
- [ ] Biết `zero_grad`.
- [ ] Biết `backward`.
- [ ] Biết `step`.

## Training và evaluation

- [ ] Hiểu epoch.
- [ ] Hiểu batch size.
- [ ] Hiểu learning rate.
- [ ] Biết `model.train()`.
- [ ] Biết `model.eval()`.
- [ ] Biết dùng `torch.no_grad()` khi evaluate.

## CNN/RNN/Transformer

- [ ] CNN phù hợp cho ảnh.
- [ ] RNN/LSTM phù hợp cho chuỗi.
- [ ] Transformer phù hợp cho ngôn ngữ và attention.
- [ ] Biết overfitting trong DL và cách giảm.
