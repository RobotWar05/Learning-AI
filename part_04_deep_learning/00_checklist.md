# Checklist Deep Learning

- [ ] Hiểu neuron nhận input và tạo output.
- [ ] Hiểu weight và bias.
- [ ] Biết activation function dùng để tạo phi tuyến.
- [ ] Hiểu loss đo sai số.
- [ ] Hiểu optimizer cập nhật trọng số.
- [ ] Hiểu epoch và batch.
- [ ] Biết tensor là cấu trúc dữ liệu chính trong PyTorch.
- [ ] Đọc được training loop cơ bản.
