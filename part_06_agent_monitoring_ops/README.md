# Phần 6: Agent Monitoring và Ops

## Mục tiêu

Hiểu tư duy quan sát, đánh giá và vận hành hệ thống AI/agent ở mức nền tảng.

## Kiến thức sẽ học

- Logs, metrics, traces.
- Latency, error rate, cost.
- Prompt/version tracking.
- Evaluation dataset.
- Human feedback.
- Deployment mindset.
- Rollback và kiểm soát thay đổi.
