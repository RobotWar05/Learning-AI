# Bài tập Agent Monitoring/Ops

1. Liệt kê các log cần lưu cho một chatbot RAG.
2. Thiết kế metric để theo dõi chất lượng chatbot.
3. Giải thích khác nhau giữa log, metric và trace.
4. Tạo bảng evaluation gồm câu hỏi, câu trả lời mong muốn, câu trả lời model.
5. Đề xuất cách phát hiện chatbot trả lời sai nguồn.

## Câu hỏi tự kiểm tra

1. Vì sao latency quan trọng?
2. Vì sao cần theo dõi cost?
3. Rollback trong hệ thống AI nghĩa là gì?
