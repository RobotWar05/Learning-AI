# Ôn nhanh Agent Monitoring/Ops

| Khái niệm | Ý nghĩa ngắn |
|---|---|
| Log | Dòng ghi sự kiện |
| Metric | Chỉ số đo theo thời gian |
| Trace | Chuỗi bước của một request |
| Latency | Thời gian phản hồi |
| Error rate | Tỷ lệ lỗi |
| Evaluation | Đánh giá chất lượng |
| Rollback | Quay về phiên bản ổn định |

## Câu hỏi nhanh

1. Log khác metric ở đâu?
2. Trace giúp debug agent như thế nào?
3. Vì sao cần lưu prompt version?
4. Khi chatbot trả lời sai, cần xem những dữ liệu nào?
5. Dashboard ops nên có các chỉ số nào?
