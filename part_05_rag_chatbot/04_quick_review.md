# Ôn nhanh RAG và Chatbot

| Khái niệm | Ý nghĩa ngắn |
|---|---|
| RAG | Kết hợp truy xuất tài liệu và sinh câu trả lời |
| Chunk | Đoạn tài liệu nhỏ |
| Embedding | Vector biểu diễn ý nghĩa text |
| Retriever | Thành phần tìm context liên quan |
| Prompt | Đầu vào có cấu trúc gửi cho model |
| Source | Nguồn dùng để kiểm chứng câu trả lời |

## Câu hỏi nhanh

1. RAG giải quyết vấn đề gì?
2. Vì sao cần chunking?
3. Khi retrieval trả về context sai thì chuyện gì xảy ra?
4. Chatbot nên làm gì khi không đủ dữ kiện?
5. Vì sao cần evaluation cho chatbot?
